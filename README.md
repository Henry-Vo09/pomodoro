# pomodoro
A digital Pomodoro timer** built with **HTML, CSS, and JavaScript**.  
It allows you to focus with customizable work/break intervals and looks like a digital clock.

## Features

- **Digital clock style** timer
- **Pastel blue & white design** for a soft, pleasant look
- Switch between **25-5 (classic)** and **50-10 (long focus)** sessions
- **Start / Pause / Reset** controls
- Auto-switches between focus and break periods
- Fully **responsive** for different screen sizes
